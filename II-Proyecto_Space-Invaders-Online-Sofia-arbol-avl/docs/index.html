Space invaders

Space Invaders is a retro shooter style video game that consists of defending
the planet from invaders who want to dominate it. The player must face different
classes of invaders and bosses with different difficulty.

The project was created in java and,for the realization of this project,
knowledge of data structures such as linked, double-linked and circular
lists was applied, mainly in the creation of enemy rows and in the player
shooting mechanics.
